# Symbol Reference

This list enumerates mathematical symbols used in the repository documentation and core equations. Update it whenever new symbols appear.

| Symbol | Definition | Introduced in |
| --- | --- | --- |
| \(A\) | Pure Kikuchi pattern from phase/material A. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(B\) | Pure Kikuchi pattern from phase/material B. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(C\) | Mixed Kikuchi pattern formed by combining \(A\) and \(B\). | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(x\) | Mixing weight for \(A\). | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(y\) | Mixing weight for \(B\), with \(y = 1 - x\). | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\hat{A}\) | Model prediction of \(A\). | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\hat{B}\) | Model prediction of \(B\). | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\hat{C}\) | Reconstructed mixture from predictions. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\hat{x}\) | Model prediction of \(x\). | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(p\) | Pixel index/location within a Kikuchi pattern. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\epsilon\) | Additive noise term in the mixing model. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\mathcal{L}\) | Training loss for reconstruction and weight supervision. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\lambda_{ab}\) | Loss weight for pure pattern reconstruction. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\lambda_{recon}\) | Loss weight for mixture reconstruction. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\lambda_x\) | Loss weight for mixing weight supervision. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\|\cdot\|_1\) | \(\ell_1\) norm used for reconstruction losses. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\mathrm{normalize}(\cdot)\) | Intensity rescaling operator for mixtures. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\mathrm{PSNR}\) | Peak signal-to-noise ratio metric. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\mathrm{SSIM}\) | Structural similarity index metric. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
| \(\text{U-Net}\) | Encoder-decoder CNN with skip connections. | [Notation and Glossary](mission_statement.md#notation-and-glossary) |
